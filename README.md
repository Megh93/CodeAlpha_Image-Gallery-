# Image Gallery

This project is a web gallery showing different foods and drinks. Anyone can filter items, like their favorites, and view images in a popup. It works on all devices and looks smooth and interactive.

## Features
- Filter images by **All**, **Food**, or **Drinks**
- **Like buttons** for each image with counter
- **Lightbox view** with next/previous navigation
- **Hover effects** for smooth interactivity
- Fully **responsive design** for mobile and desktop
- Clean and professional layout

## How to Use
1. Clone or download the repository.
2. Make sure the `images` folder contains all 8 images with these names:
   - `pizza.jpg`
   - `pasta.jpg`
   - `mango-mocktail.jpg`
   - `iced-coffee.jpg`
   - `strawberry-donuts.jpg`
   - `chocolate-cake.jpg`
   - `pancakes.jpg`
   - `quesadilla.jpg`
3. Open `index.html` in your browser.
4. Click on images to view them in the lightbox popup.
5. Click ❤ to like your favorite items.
6. Use the filter buttons to see Food or Drinks only.
